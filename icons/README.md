# 图标文件说明

由于技术限制，实际的图标文件需要您自行添加。请在此文件夹中放置以下图标文件：

## 需要的图标文件

- `icon16.png` - 16x16像素，用于扩展管理页面
- `icon48.png` - 48x48像素，用于扩展管理页面和通知
- `icon128.png` - 128x128像素，用于Chrome Web Store

## 图标设计建议

- 使用网球主题图标（🎾）
- 背景色建议：绿色或蓝色
- 风格：简洁现代
- 格式：PNG，透明背景

## 临时解决方案

如果暂时没有图标文件，扩展依然可以正常工作，只是会显示默认的扩展图标。
